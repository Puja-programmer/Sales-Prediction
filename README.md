# Sales Prediction using Machine Learning

## Overview

This project predicts sales based on advertising data using machine learning techniques. It analyzes how different advertising channels influence sales and builds models to forecast outcomes.

## Objectives

* Analyze the relationship between advertising spend and sales
* Build predictive models using machine learning
* Compare model performance

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

The dataset includes:

* TV advertising spend
* Radio advertising spend
* Newspaper advertising spend
* Sales

## Workflow

1. Data loading and preprocessing
2. Exploratory data analysis
3. Model building (Linear Regression, Random Forest)
4. Model evaluation using MAE and R² score
5. Visualization of results

## Results

* Built models to predict sales
* Compared performance of different algorithms
* Identified important features affecting sales
